# wfprov:Artifact

## Description

An **Artifact** represents a data entity that was used as input or generated as output during a workflow execution. Artifacts are the data products consumed or produced by process runs.

## Relations

- Can be linked to a `wfdesc:Input` or `wfdesc:Output` via `describedByParameter`
- Can be generated by a `wfprov:ProcessRun` via `wasOutputFrom`
- Can be used by a `wfprov:ProcessRun` via `prov:used`
